## サンプルの動かしかた

### C++

```
$ cd c++
$ mkdir build
$ cd build
$ cmake .. -DCMAKE_BUILD_TYPE=Release
$ make
$ ./sample
```

### Python

```
$ cd python
$ pip3 install -r requirements.txt 
$ python -v
```
